# Кулинарная студия «Шеф-стол» — MVP клиентской части

Интерактивное веб-приложение (Single Page Application) для бронирования кулинарных мастер-классов. Прототип разработан в рамках учебной практики и полностью готов к тестированию без необходимости развертывания бэкенда.

## Тема проекта
Автоматизация записи клиентов на кулинарные мастер-классы на 7 дней вперед с учетом доступности мест, возможных отмен мероприятий со стороны студии, а также сбором обратной связи от участников.

---

## Что внутри: функционал и интерфейс

Интерфейс разделен на две основные вкладки в шапке приложения:

### 1. Вкладка «Расписание классов»
Здесь отображается лента доступных мастер-классов. Карточки имеют три динамических состояния:
* **«Есть места»** — яркая кнопка **[Записаться]**. Открывает модальное окно.
* **«Мест нет»** — кнопка заблокирована. Сигнализирует о полном заполнении группы.
* **«Отменён»** — карточка окрашивается в серый цвет, выводится плашка с причиной форс-мажора, кнопка **[Класс отменён]** полностью отключена.

**Логика модального окна записи:**
* Отображает базовую стоимость.
* Содержит интерактивный чекбокс **«Аренда фартука и ножей (+500 руб)»**. При клике итоговая стоимость мгновенно пересчитывается на экране.
* Кнопка **[Подтвердить запись]** — списывает 1 место с главного экрана и переносит запись в личный кабинет.

### 2. Вкладка «Мои бронирования»
Личный кабинет пользователя, где отображаются все успешные записи. Для ручного тестирования здесь доступны две механики:
* Кнопка **[Отменить]** — мгновенно удаляет бронь, возвращает место в расписание на главном экране и активирует кнопку записи, если класс был заполнен.
* Кнопка **[Завершить класс]** — эмулирует «машину времени» (прохождение класса). Переводит статус в «Завершен» и открывает модальное окно с **5-звездочным интерактивным рейтингом** для оценки шеф-повара.

## Как запустить проект

Приложение спроектировано как легковесный SPA-прототип, поэтому для запуска не требуются сборщики (Webpack/Vite) или локальный сервер:

1. Склонируйте репозиторий или скачайте файлы в одну папку.
2. Убедитесь, что файлы `index.html`, `style.css` и `script.js` лежат в одной директории.
3. Просто **дважды кликните по файлу `index.html`**, чтобы открыть приложение в любом современном браузере.
