# User Stories (Пользовательские истории)

> Этап 2. Пользовательские истории с критериями приемки (Acceptance Criteria) для роли «Клиент».

| ID | Роль | Я хочу... (Действие) | Чтобы... (Ценность) | Критерии приемки (Acceptance Criteria) | Связь с FR |
| :-- | :-- | :-- | :-- | :-- | :-- |
| US-1 | Клиент | Быстро авторизоваться по имени и телефону. | Не тратить время на создание сложных паролей и сразу перейти к бронированию. | После ввода имени и телефона данные сохраняются в объекте сессии, интерфейс переключается в авторизованное состояние. | FR-1, FR-2 |
| US-2 | Клиент | Видеть актуальный список кулинарных классов на 7 дней вперед. | Выбрать интересное меню и удобное время для посещения студии. | Лента содержит карточки с названиями, датами, ценой и количеством мест, отсортированные по хронологическому порядку. | FR-9 |
| US-3 | Клиент | Фильтровать список классов по шеф-поварам. | Записываться на мастер-классы конкретных преподавателей. | При выборе шеф-повара из выпадающего списка на экране остаются только его классы; при сбросе фильтра отображаются все классы. | FR-38 |
| US-4 | Клиент | Видеть рейтинг шеф-повара в карточке класса. | Оценить профессионализм кулинара перед совершением записи. | Рядом с именем шеф-повара выводится среднее числовое значение его рейтинга `Chef.rating` из Mock-данных. | FR-9a |
| US-5 | Клиент | Добавлять опцию аренды фартука и ножей при записи. | Не возить с собой личную домашнюю форму и инвентарь. | Включение чекбокса динамически увеличивает итоговую сумму бронирования на величину `rental_equipment_price`. | FR-11, FR-45 |
| US-6 | Клиент | Видеть текстовую причину отмены мастер-класса. | Знать, почему занятие не состоится, если студия его отменила. | На карточках со статусом `canceled_by_studio` отображается блок текста с содержимым поля `cancel_reason`. | FR-44 |
| US-7 | Клиент | Самостоятельно отменять записи в своем личном кабинете. | Освободить место на кухне для других людей, если мои планы изменились. | При нажатии кнопки отмены запись меняет статус, а количество доступных мест `available_places` у соответствующего класса возрастает на 1. | FR-16 |
| US-8 | Клиент | Оценивать работу шеф-повара после завершения мастер-класса. | Оставить обратную связь администрации о качестве мероприятия. | В списке прошедших бронирований доступно интерактивное поле для записи значения `chef_rating` (от 1 до 5). | FR-50 |
